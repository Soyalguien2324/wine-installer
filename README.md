# wine-installer
Un simple instalador de wine y Steam. 
Para descargarlo:
```bash
git clone https://github.com/Soyalguien2324/wine-installer
cd wine-installer
chmod a+x main.sh
clear
./main.sh
```
Cuando os salga esta pantalla, pulsais donde os pone windows el que sea y poneis el Windows 10
